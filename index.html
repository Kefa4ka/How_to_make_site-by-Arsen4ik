Як створювати сайти у Visual Studio Code

Що таке Visual Studio Code?

Visual Studio Code (VS Code) - це безкоштовний редактор коду від Microsoft, який підтримує багато мов програмування та має велику кількість розширень. VS Code особливо популярний серед веб-розробників завдяки своїй гнучкості та потужним функціям.

Створення проекту

  1. Відкрийте VS Code
  2. Виберіть "File" > "Open Folder"
  3. Створіть нову папку для вашого проекту
  4. Створіть файли index.html, style.css, і script.js

Структура вашого проекту може виглядати так:


my-website/
│
├── index.html
├── style.css
└── script.js

Написання коду

VS Code надає підсвічування синтаксису, автодоповнення та інші корисні функції для написання HTML, CSS та JavaScript. Ось базовий приклад HTML-файлу:


<!DOCTYPE html>
<html lang="uk">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Моя перша веб-сторінка</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Ласкаво просимо на мою веб-сторінку!</h1>
    <p>Це мій перший сайт, створений за допомогою Visual Studio Code.</p>
    <script src="script.js"></script>
</body>
</html>

Розбір коду:

  • <!DOCTYPE html> - оголошення типу документа для HTML5
  • <html lang="uk"> - кореневий елемент HTML з атрибутом мови
  • <head> - містить метадані та посилання на зовнішні ресурси
  • <meta charset="UTF-8"> - встановлює кодування символів
  • <meta name="viewport"> - налаштовує відображення на мобільних пристроях
  • <title> - встановлює заголовок сторінки
  • <link> - підключає зовнішній CSS-файл
  • <body> - містить видимий вміст сторінки
  • <script> - підключає зовнішній JavaScript-файл

Використання терміналу

VS Code має вбудований термінал. Ви можете використовувати його для запуску команд npm, git та інших інструментів командного рядка. Щоб відкрити термінал, натисніть Ctrl+` (або View > Terminal у меню).

Приклад команд:


# Ініціалізація git репозиторію
git init

# Встановлення пакету через npm
npm install lodash

# Запуск локального сервера (якщо встановлено розширення Live Server)
live-server

Розширення

Встановіть корисні розширення для покращення вашого досвіду розробки:

  • Live Server: Дозволяє запускати локальний сервер з автоматичним оновленням сторінки
  • Prettier: Автоматично форматує ваш код
  • ESLint: Допомагає знаходити та виправляти проблеми у JavaScript коді
  • Auto Rename Tag: Автоматично перейменовує парні HTML/XML теги

Щоб встановити розширення, перейдіть у вкладку Extensions (Ctrl+Shift+X) та знайдіть потрібне розширення.

Корисні комбінації клавіш

  • Ctrl+S: Зберегти файл
  • Ctrl+/: Закоментувати/розкоментувати вибрані рядки
  • Alt+Up/Down: Перемістити рядок вгору/вниз
  • Ctrl+F: Пошук
  • Ctrl+H: Заміна
  • F1 або Ctrl+Shift+P: Відкрити командну палітру
